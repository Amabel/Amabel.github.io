Java: 使用 Deque 模拟栈和队列

April 1, 2018 in java, lib

在编程时很多时候需要用到栈和队列，自己编码实现比较麻烦。本文介绍如何用 JDK 自带的类来实现栈和队列。

关于 Deque

JDK 中使用Deque (Double ended queue)接口来模拟栈和队列的相关操作。常用类如LinkedList, ArrayDeque都实现了Deque接口。

Deque 的用法

引入包：

import java.util.Deque;
import java.util.LinkedList;

声明时要将LinkedList当作Deque：

Deque<Integer> stack = new LinkedList<>();
stack.addFirst(3); // push
stack.removeFirst; // pop

Deque 接口的方法

Deque接口提供对集合中的首元素和末尾元素的相关操作。对于一种操作有相应的两种方法，这两种方法的效果基本相同，但是在操作失败时一个会抛出异常(add / remove / get)，另一个会返回false(offer / poll / peek)。
